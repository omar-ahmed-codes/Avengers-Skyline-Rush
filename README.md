# Avengers: Skyline Rush

An Avengers-themed endless 3D road-crossing game inspired by Crossy Road, built with Three.js.

## Live Game

- Production: https://avengers-skyline-rush.vercel.app

## Highlights

- Smooth 3D WebGL gameplay with stylized Lego/Roblox-inspired Avengers heroes and villain traffic
- Endless forward progression with camera-follow scrolling (no hard reset while alive)
- Avengers city backdrop with tower skyline atmosphere
- Hero picker, in-game Play/Restart overlay, and Pause/Sound controls
- Persistent high-score leaderboard stored in browser `localStorage` (top 5 runs)
- Synthwave-style procedural soundtrack and SFX generated in-game

## Play Controls

- Move: `Arrow Keys` or `W A S D`
- Pause/Resume: `Space` or `Pause` button
- Mobile/Desktop touch: on-screen directional controls

## Run Locally

1. From the project root, start a static server:
   - `python3 -m http.server 8081`
2. Open:
   - `http://localhost:8081`
